Rubiks Cube

Run from Command line

Argparser Controls:

-d : Dicing of Cube (Default 3)
-t : Rate of turn (Default 1 second)
-m: Number of moves (Default 15)

Keyboard User Controls:

Esc: Quit the program
Left Click: Rotate the cube
S: Scramble the cube
F: Solve a scrambled cube
R: Reset the cube without animation
C: User input single move for the cube
O: Save the current state of the cube
L: Load a previously saved state of the cube
N: Solve the next move of the cube
